# pdf-notebook

React + Tailwind CSS による最小限の構成

## 開発環境のセットアップ

```bash
npm install
```

## 開発サーバーの起動

```bash
npm run dev
```

## ビルド

```bash
npm run build
```

## プレビュー

```bash
npm run preview
```

## 技術構成

- **React 18** - UI フレームワーク
- **Tailwind CSS 3** - ユーティリティファーストのCSS フレームワーク  
- **Vite 5** - 高速なビルドツール
- **PostCSS** - CSS後処理
- **Autoprefixer** - ベンダープレフィックスの自動付与
